mac-osx-free-command-in-python
==============================

A simple python script of "free" command implementation with colorful output for Mac OS X

usage
==============================

<pre>
free [-b|-k|-m|-g]
  -b,-k,-m,-g show output in bytes, KB, MB, or GB
  default output in KiloBytes
</pre>
